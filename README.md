# 🔐 AuthPro – Advanced Login & Authentication System

A modern, secure, and visually polished authentication system built using **PHP, MySQL, HTML, CSS, and JavaScript**.

This project is demonstrating real-world backend concepts like session handling, password hashing, user management, and activity tracking — combined with a modern UI/UX design.

---

## 🚀 Features

### 🔑 Authentication
- User Signup with validation
- Secure Login system
- Password hashing using `password_hash()`
- Password verification using `password_verify()`
- Session-based authentication

### 👤 User Management
- Profile view page
- Edit profile (name & email)
- Change password functionality
- Logged-in state detection

### 📊 Dashboard
- Personalized dashboard
- User details (email, role, join date)
- Last login tracking
- Recent login/logout activity logs

### 🔒 Security Features
- Session regeneration (prevents session fixation)
- Input sanitization
- Basic validation (email, password length, matching passwords)
- Session timeout (auto logout after inactivity)

### 📁 Activity Tracking
- Logs user login and logout events
- Displays recent activity in dashboard

### 🎨 UI / UX
- Gradient modern UI
- Animated buttons and hover effects
- Cursor glow effect
- Glassmorphism cards
- Responsive design (mobile-friendly)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP (Apache)
